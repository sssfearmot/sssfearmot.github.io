    test01

    overflow : 요소의 박스에 내용이 더 길 때 어떻게 보일지 선택하는 속성
    + visible 기본값으로 내용이 더 길어도 그대로 보임 
    + hidden 내용이 넘치면 자름. 자른 부분은 보이지 않음, 
    + scroll 내용이 넘치지 않아도 항상 스크롤바가 보임, 
    + auto 내용이 잘릴 때만 스크롤바가 보임

    z-index : 어느 객체가 앞으로 나오고, 뒤에 나올지 배치 순서를 결정하는 속성. 
    + position (relative, absolute, fixed) 속성이 적용된 요소에서만 작동. 
    + auto 기본값으로 z-index를 지정하지 않은 것과 같음 
    + number 배치 순서를 결정, 숫자가 낮을수록 뒤에 배치되며 숫자가 높을수록 앞에 나옴 (음수도 가능) 
    + initial 기본값으로 설정

  • 포지션01

    position01

    position:static을 제외한 나머지 속성값들은 top, left, right, bottom  와 같이 어느 위치에 요소를 위치시킬 건지 지정할 수 있다.
    + static은 position 속성의 기본값, 요소를 문서 흐름에 맞추어 배치
    + relative 이전 요소(주로 부모 요소)에 자연스럽게 연결하여 위치를 지정
    + absolute 원하는 위치를 지정해 배치
    + fixed 지정한 위치에 고정하여 배치
    + sticky 위치에 따라서 동작방식이 달라짐.

    nth-child() 속성
    + 홀수번째 엘리먼트에 적용 : nth-child(odd)
    + 짝수번째 엘리먼트에 적용 : nth-child(even)
    + 2n번째에 해당하는 엘리먼트 : nth-child(2n)

    Data 속성: 'data-'로 시작하는 속성
    + 이러한 데이터 속성은 특정한 데이터를 DOM 요소에 저장해두기 위함이 목적
    + ex) data-value="001"
    + 속성명은 data-value이지만 HTML 표준 속성인 value 속성과는 상관이 없다.
    임의로 만들어진 속성인 사용자 지정 데이터이기 때문이다.

    + attr(해당 속성의 속성값 표시) content 속성 + Data 속성

    • justify-content : 가로 축을 기준으로 좌우에 대한 정렬을 관장
    + flex-start 요소들을 컨테이너의 왼쪽으로 정렬
    + flex-end 컨테이너의 우측으로 정렬
    + center 컨테이너의 중앙으로 정렬
    + space-between 요소들 사이에 동일한 간격을 둔다
    + space-around 요소들 주위에 동일한 간격을 둔다

    text-align : block 요소 안에 있는 inline 요소를 정렬
    + left 왼쪽 정렬
    + right 오른쪽 정렬
    + center 중앙 정렬
    + justify 왼쪽과 오른쪽 열에 맞추어 화면을 늘어뜨림
        
    *주의사항*
    1. block 요소에만 text-align 속성을 적용할 수 있다.
    2. 정렬되는 것은 block 요소 안에 있는 inline 요소만 가능
    3. 2번에서 inline 요소란 text뿐 아니라 이미지 등도 포함
    inline 요소 : 줄 속에 끼워 넣는 요소
    ex) span, b, a, img 태그 등
    block 요소 : 해당 요소가 속한 줄은 가로 화면 전체 100%를 차지
    ex) p, div, ol, ul, table 등
