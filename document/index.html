position

  1. position01

    absolute 활용

    absolute 속성값은 브라우저가 문서의 흐름과 상관없이 left, right, top, bottom 속성값을 이용하여 요소를 위치시키는 속성값
    + 부모 엘리먼트 위에 겹쳐서 배치 가능
            
    transform은 두번 쓰지 않는다
    transform은 회전, 크기 조절, 기울이기, 이동 효과 등을 부여할 때 사용
    2022.05.19 Keyword : position, absolute, transform
  2. position02

    fixed 활용

    fixed는 브라우저 창을 어디로 하더라도 계속 고정되어 표시됨(기준점 : 브라우저 왼쪽 위 꼭지점)
    + absolute 속성값과 마찬가지로 문서의 흐름과 상관없이 위치를 좌표로 결정
            
    transition은 CSS 속성을 변경할 때 두 가지 상태 사이에 일어나는 변화(애니메이션)를 커스터마이징 할 때 사용
    + 속성 변경이 즉시 영향을 미치게 하는 대신, 그 속성의 변화가 일정 기간에 걸쳐 일어나도록 하는 것
    2022.05.19 Keyword : position, fixed, transition
  3. position03

    JS 사용, hover 활용, 폰트, 아이콘

    hover는 마우스를 위에 올렸을때 반응/ figure:hover = 사진 따로 반응
    overflow는 박스 밖으로 넘치는걸 막음 / flex:1 = 사진을 균등하게 배치
    content 선택한 요소의 내용 앞이나 뒤에 콘텐트를 붙임
    inline-block으로 지정된 엘리먼트는 전후 줄바꿈 없이 한 줄에 다른 엘리먼트들과 나란히 배치
    +(width, height, margin, padding 지정 가능)
            
    id는 하나만 사용 가능, id를 호출할때는 # (ex. #event)
            
    폰트 사용 : 구글 웹 폰트
    아이콘 사용 : XEIcon
    2022.05.20 Keyword : JS, hover, font, icon
